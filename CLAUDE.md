# Clouvel

> PRD 없으면 코딩 없다.

---

## 개요

Clouvel은 바이브코딩 프로세스를 강제하는 MCP 서버입니다.

문서 없이 코딩 시작? 차단됩니다.

---

## 기본 사용법

```bash
# 설치
pip install clouvel

# Claude Code에서 사용
claude --mcp clouvel
```

---

## 핵심 도구

| 도구 | 설명 |
|------|------|
| `can_code` | 코딩 가능 여부 검사 |
| `get_progress` | 진행 상황 확인 |
| `get_goal` | 프로젝트 목표 리마인드 |

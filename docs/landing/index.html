설치 가이드

PyPI에서 바로 설치, Claude Code와 연동

Personal 개인 사용자

# 1. 설치

pip install clouvel-pro

# 2. Claude Code 설정 (~/.config/claude/claude_desktop_config.json)

{
  "mcpServers": {
    "clouvel-pro": {
      "command": "clouvel-pro"
    }
  }
}

# 3. Claude Code에서 라이선스 활성화

> activate_license(license_key="YOUR-LICENSE-KEY")

Team 팀 사용자

# 1. 모든 팀원이 각자 PC에 설치

pip install clouvel-pro

# 2. 오너(구매자)가 먼저 라이선스 활성화

> activate_license(license_key="TEAM-LICENSE-KEY")

# 3. 오너가 팀원 초대

> team_invite(email="member@company.com", role="member")

# 4. 팀원도 같은 라이선스 키로 활성화

> activate_license(license_key="TEAM-LICENSE-KEY")

# 5. 에러 패턴 팀 공유

> sync_team_errors(project_path=".") # 로컬 → 팀

> apply_team_rules(project_path=".") # 팀 → 로컬

VS Code 사용자: Claude Code 확장 설치 후 동일하게 사용

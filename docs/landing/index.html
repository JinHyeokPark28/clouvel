문제

바이브코딩의 함정

"AI야 이거 만들어줘" → 코드 생성 → 버그 → 수정 → 또 버그 → 처음부터 다시

Clouvel 방식

PRD 작성 → 문서 검증 → 코딩 허용 → 체계적 개발 → 완성

작동 방식

1

can_code 호출

코드 작성 전 docs 폴더 확인. PRD, 아키텍처, API 문서 체크.

2

문서 없으면 코딩 차단

"코딩 금지" 반환. PRD 작성 가이드 제공.

3

문서 완성 → 코딩 허용

"코딩 가능" 반환. 체계적 개발 시작.

기능

📋

문서 강제
